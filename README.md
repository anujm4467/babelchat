#### Dev environment

 * git clone https://github.com/danielesteban/babelchat.git
 * write your `GOOGLE_CLIENT_ID` and `GOOGLE_CLIENT_SECRET` into `api/.env`
 * yarn install
 * npm start (or press F5 in vscode)
